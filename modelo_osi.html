Bem-vindo ao primeiro capítulo da nossa jornada pela certificação CCNA! Neste módulo, vamos desmistificar os conceitos mais fundamentais que formam a base de todas as redes de computadores modernas. Começaremos entendendo o que é uma rede, qual seu propósito e por que ela é tão essencial no mundo digital de hoje.

O Que é uma Rede?

Em sua forma mais simples, uma rede de computadores é um conjunto de dois ou mais dispositivos (como computadores, servidores, celulares ou impressoras ) conectados entre si para compartilhar recursos e trocar informações. O recurso compartilhado pode ser um arquivo, uma conexão com a internet, uma impressora, etc.

Pense na rede como o sistema circulatório do mundo digital. Assim como o sangue transporta oxigênio e nutrientes pelo corpo, a rede transporta dados, permitindo a comunicação e a colaboração em escala global.

Modelos de Referência: OSI e TCP/IP

Para garantir que dispositivos de diferentes fabricantes possam se comunicar, a indústria criou modelos de referência. Os dois mais importantes são o Modelo OSI e o Modelo TCP/IP. Eles dividem a complexa tarefa da comunicação em camadas, onde cada camada tem uma função específica.

O Modelo OSI tem 7 camadas, enquanto o TCP/IP, mais prático, tem 4 (ou 5, dependendo da interpretação). Para o CCNA, é crucial entender ambos.

Exemplo de Bloco de Código

Ao configurar um roteador, você usará uma interface de linha de comando (CLI). Um comando simples para ver as interfaces de rede seria: 

Router> show ip interface brief

Este comando, show ip interface brief, é um dos mais utilizados no dia a dia de um analista de redes.
