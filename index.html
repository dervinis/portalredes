Bem-vindo!

Este é o seu hub de conhecimento para aprender sobre redes de computadores, do básico ao avançado, com uma abordagem totalmente prática.

O objetivo deste projeto é fornecer material de estudo gratuito e de alta qualidade para todos que desejam ingressar na área de infraestrutura e redes.

Trilha de Estudos: CCNA 200-301

Atualmente, o foco principal é a criação de conteúdo para a certificação Cisco CCNA. Abaixo estão os tópicos em desenvolvimento:

  • Fundamentos de Redes: O que são switches, roteadores e o modelo OSI.
  • Endereçamento IP: Dominando IPv4, sub-redes e VLSM.
  • Protocolos de Roteamento: Configuração prática de OSPF.

Exemplo de configuração de interface em um roteador Cisco:

Router> enable
Router# configure terminal
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
